# Cherry URL Shortener ✂️
Red, Cherry 🍒 Themed Web Service for Shortening urls

# Technologies 🧾
* Node.js 
  * koa
  * shortid
  * mongoose
* Tailwind CSS

# Quick Start 🚀
```git clone https://github.com/irevenko/url-shortener.git``` <br>
```cd url-shortener``` <br>
```npm install``` <br>
```touch .env (Add your variables)``` <br>
```npm start```
